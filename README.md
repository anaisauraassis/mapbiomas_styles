# mapbiomas_styles
MapBiomas Style aplica automaticamente a simbologia de uso e cobertura do solo do MapBiomas na camada ativa do QGIS com base no campo de identificação "DN" ou "Uso". Além disso, calcula a área de cada categoria em hectares (ha) e km², e mostra a porcentagem de cobertura relativa, facilitando análises de geoprocessamento e planejamento ambiental.
